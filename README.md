# Project Overview

This project is a modern and responsive personal portfolio website developed using **HTML** and **CSS**. The portfolio is designed with a premium dark-themed user interface inspired by modern developer portfolio designs.

The website showcases personal information, technical skills, projects, and contact details in an attractive and professional manner.

The design includes:

* Glassmorphism UI
* Animated blur background effects
* Responsive layout
* Interactive buttons
* Social media links
* Skill progress bars
* Project showcase cards

---
